# data/srd-5-1

Carpeta para contenido SRD 5.1 estructurado.

No es necesario cargar todo desde el principio. Recomendacion:

1. Crear schemas Zod.
2. Crear seeds pequenos.
3. Validar importacion.
4. Ampliar catalogo progresivamente.

Recordatorio de atribucion:

Esta obra incluye materiales extraidos del Documento de referencia del sistema 5.1 ("SRD 5.1") de Wizards of the Coast LLC, que esta disponible en https://dnd.wizards.com/es/resources/systems-reference-document.
