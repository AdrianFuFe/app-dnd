# App DnD - Documentacion ampliada para Codex

Este paquete amplia la documentacion del proyecto con:

- Integracion de contenido reutilizable del SRD 5.1.
- Comparativa resumida entre Manual del Jugador 5e y SRD 5.1.
- Modelo para contenido manual/privado.
- Sistema de permisos y visibilidad.
- Guia para cargar contenido desde archivos del proyecto.
- Roadmap de implementacion.

## Uso previsto

Copiar estos documentos en `docs/` del proyecto y pasar el prompt de `prompts/01_PROMPT_CODEX_SRD_PERMISOS_CONTENIDO.md` a Codex en VS Code.

## Fuentes y limites

El SRD 5.1 esta publicado bajo Creative Commons Atribucion 4.0. Se puede usar como base reutilizable respetando la atribucion requerida.

Este paquete no reconstruye el Manual del Jugador completo ni reproduce tablas o descripciones protegidas. Para contenido que no este en SRD, la app debe permitir carga manual como contenido privado del usuario.

## Atribucion SRD 5.1

Esta obra incluye materiales extraidos del Documento de referencia del sistema 5.1 ("SRD 5.1") de Wizards of the Coast LLC, que esta disponible en https://dnd.wizards.com/es/resources/systems-reference-document. El SRD 5.1 tiene la licencia Creative Commons Atribucion/Reconocimiento 4.0 Licencia Publica Internacional, que esta disponible en https://creativecommons.org/licenses/by/4.0/legalcode.es.
